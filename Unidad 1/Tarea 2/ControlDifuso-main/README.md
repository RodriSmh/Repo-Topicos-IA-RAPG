# ControlDifuso
Desarrollo de un sisitema de control difuso en pyrhon para el calculo de probablidades

Para el uso de este codigo tenemos 5 funciones de memebresia en cuanto a dinero ahorrado en el banco las cuales junto a las 3 funciones de membresia de salario sus nombres se especifican en el codigo el cual se presenta en este repositorio, para el calculo de alguna probablidad solo es necesario el cambio del numero el cual hacer referecnia al ahorro y el salario, estos parametros se pueden cambiar en la parte final del codigo donde se indica la funcion controlador.input , y se hace el calculod e la probablidad respecto a lo estipulado previamente
